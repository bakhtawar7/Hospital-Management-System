# Hospital Management System (C++ PF Project)

## Project Description
The Hospital Management System is a C++ project developed using programming fundamentals concepts. It simulates basic hospital operations such as managing patients, doctors, appointments, and records using a console-based interface.

## Features
- Patient registration and management
- Doctor record management
- Appointment scheduling
- Basic billing system
- Data storage using file handling
- Menu-driven interface

## Concepts Used
- C++ Functions
- Arrays
- File Handling
- Control Structures (if-else, loops)
- Basic OOP concepts (if applied)

## How to Run
1. Open the project in any C++ IDE (Code::Blocks / VS Code / Dev C++)
2. Compile the source file
3. Run the program in console
4. Use menu options to interact with the system
